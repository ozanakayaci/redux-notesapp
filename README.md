## Getting Started

### Prerequisites

- npm
  ```sh
  npm install npm@latest -g
  ```

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/ozanakayaci/redux-notesapp.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Runs the app in the development mode.

   ```sh
   npm start
   ```
